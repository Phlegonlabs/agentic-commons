# Support

## Scope

Agentic Commons is maintained on a best-effort basis.

- No SLA is provided.
- Response times vary by maintainer availability.

## Where to Ask

- Usage questions: GitHub Issues
- Bug reports: GitHub Issues using the bug template
- Feature requests: GitHub Issues using the feature template

## Security Issues

Do not report vulnerabilities in public issues.

Use GitHub Security Advisories as described in `SECURITY.md`.

## Before Opening an Issue

Please include:

- Environment details (OS, Node version, npm version)
- Exact command and output
- Reproduction steps
- Expected behavior vs actual behavior
